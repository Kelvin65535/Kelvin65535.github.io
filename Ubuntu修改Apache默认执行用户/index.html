Ubuntu修改Apache默认执行用户

发表于 2017-09-01 更新于 2019-11-27 阅读次数： Disqus：

前言

Ubuntu的Apache默认执行用户是www-data，用户组是www-data，但是由于www-data为低权限用户，有时会出现使用php读取文件、执行脚本时出现权限不够的情况，因此可以通过修改Apache的配置，达到使用其他用户执行Apache的效果。

适用版本：Apache 2.4.7
